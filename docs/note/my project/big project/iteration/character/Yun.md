### 特质:

#### 暂时性兴趣: 
最初, 由于兴趣和好奇, 浮芸能快速高效地完成某项事务。但随着时间推移, 大部分事务的效率和质量会下降。然而, 经过一段时间后, 它们能慢慢恢复。"效率与质量 - 时间关系图"类似于函数 f(x)=sin x 的图像。


#### 警觉性: 
长期执行危险任务的身体通常保持着高度警觉。每当异常情况发生时, 芸几乎都能立即察觉。


#### 厌战: 
厌倦战斗,浮芸不擅长战斗, 因此他总是尽量避免争斗。浮芸更喜欢享受生活而非做其他事, 所以在休息时他会更有效率。


#### 双重意识: 
共存于同一身体内的两个意识相处融洽。同一时间只有一个意识控制身体, 但这不影响他们之间的交流。

::: collapse 虚拟现实折叠:
拥有此项能力后,使用者可以无代价创造至多两个虚拟世界,并无代价地对世界进行修改,包括但不仅限于创造,修改实体信息和数据,清除对象等

虚拟折叠:将虚拟世界中的改动同步至现实世界

现实折叠:将现实世界的事物强制折叠进虚拟世界

虚实再折叠:将事物折叠进虚拟世界并执行一定操作,再折叠进现实世界

数字转化:将一定范围内的场景数字化,并执行一定操作,例子如下:

基础操作:如操作系统般拖动事物,左键和右键,删除和复制剪切等

视频方面:暂停特定事物,倍速播放事物,修改事物音量等

图像方面:画笔,橡皮,等以及图层修改(不同图层的事物无法互相产生影响)等方面

影像方面:修改速度,影重(类似力量),透明度等

连接(可随意开关):

有线连接:所有事物无法使用超自然能力(包括Yun),限制事物的直接连接

无线连接:事物无法进行通讯以及远距离操作(包括超自然能力的远程操作)
:::

#### 武器:

##### 风雨千衾:

###### 材料: 
提取自 E.S-C, 高强度合金

###### 属性: 
多重升级, 双重形态, 附灵机械, 二次改造, 消耗与补充

###### 信息:
一件优雅的武器, 分为两种不同的形态: 扇与伞。

扇: 一把精致的扇子, 扇叶可以分离并发射, 并能增强超常能力的冲击效果。这使得使用者能够方便地在远距离使用能力。扇叶和扇柄都包含可自由弹射的长刃。这些特性使其拥有有效的远程攻击能力, 并可在近战时如剑般挥舞。锋利的长刃以及可折叠、可展开的扇叶使这种形态能够进行多样化的战斗。整体材质使得展开的扇子具备出色的防御能力。(实际上, 两种形态都具备多样化的用途和防御能力, 只是各有所长。)

伞: 基于展开的扇叶结构, 伞形态拥有非常广阔的防御面积。此外, 向内的扇叶可以减少正面受击面积, 配合弹射出的尖刺, 伞可以像长矛一样使用。面向外侧的扇叶可以沿着伞面朝向的方向发射出去。

::: warning 精神需求: 高
:::

::: tip 特殊效果:
疼痛转化: 无论此武器对任何生物造成何种伤害, 都能为使用者恢复精神力.
扇叶生成: 使用者可以通过消耗精神力来生成扇叶.
消耗: 中低
:::


::: note 特定效果/用法:
对于芸: 基于"空间操控"能力, 芸可以改变扇叶发射的方向. 并且芸可以分离扇叶使其环绕自身, 然后进行发射. 这种用法能使芸储存或使用更多扇叶.
:::
