Memorandum:
机制——芸——拍照:游戏项目中，玩家常驻一个拍照”按钮，保存镜头画面（基础参数）功能包括滤镜，影象参数，动作等。也可使用角色能力，支持设定延迟连续任务）。在某些特定场景拍摄，会附上一首Fuyun的诗。这些照片达到一定数量后，再与离言交互，即可得到“虚拟现实折叠”能力。无论在何处拍照，玩家都可以以芸凌的名义附上一些文字（诗，词，句，字…都可以）。
成就—离言:离言（即 Leavingwords）是一位对现实世界失望的人物。在阴差阳错之下，得到了碎片流，成为飞升者计划的一部分。她是游戏项目中驱动成就系统的主体，通过虚拟世界的记忆读取和推演得到成就列表。成就的显示以对话（诗意语言）为主，界面交互为辅。会隐隐显示已达成成就附近的未完成成就。

游戏—赠品—壁纸:完成芸的各个结局或某些成就会附赠一张相关的交互式动态壁纸。玩家在游戏中拍的照/录的屏都可以设置为壁纸。
游戏—营销—预览版游戏在发售之前会免费发出预览版。每一次游戏在更新内容前，预览版都会临时开放部分新内容。当然，正式版也有预览接口，接口持续到正式版更新，且支持内容继承。选变续作:基于项目的元叙事框架，除了 Iteration1以外，超位者还会有其他模拟实验: :Itercation2.3.每个模拟实验都可以加入不同的环境变量（规则，存在…）衍生出丰富的，无限的用于叙事的环境（一般是三维世界）。而项目组随时可以将视角切换到任一试验场，发布作品。
